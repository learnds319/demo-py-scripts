Migration Strategy: Hybrid SSH Pattern

The Rules Loader consolidates business rules from an On-Premises Oracle database. To mitigate high risks associated with rewriting complex legacy logic and establishing new Oracle connectivity from the cloud, we will retain the Perl script On-Premises. Orchestration will be modernized by triggering the script via Secure SSH Commands from AWS Step Functions, using the Token Callback Pattern for reliable monitoring.

import sys, subprocess, boto3, argparse # 1. Parse Token passed via SSH command line parser = argparse.ArgumentParser() parser.add_argument("--token", required=True, help="Step Functions Task Token") parser.add_argument("--script", default="/opt/legacy/rules_loader.pl") args = parser.parse_args() # Initialize AWS Client (uses local IAM Roles Anywhere or Credentials) client = boto3.client('stepfunctions', region_name='us-east-1') try: # 2. Run the Legacy Perl Script print(f"Starting Rules Loader: {args.script}") subprocess.run(["perl", args.script], check=True) # 3. Success Callback: Tell AWS we are done print("Job Succeeded. Sending Callback...") client.send_task_success( taskToken=args.token, output='{"status": "success"}' ) except Exception as e: # 4. Failure Callback: Tell AWS we failed print(f"Job Failed: {e}") client.send_task_failure( taskToken=args.token, error="WrapperError", cause=str(e) )

Operational Flow: "Life of a Rules Job"

1

Trigger

EventBridge

EventBridge fires the scheduled "Start Execution" signal every 4 hours. The Step Function workflow wakes up.

2

Token Generation

Step Functions

Step Functions creates a unique Token (e.g., TOKEN_XYZ_123...) and enters the Wait State. It effectively pauses here.

3

The Handoff

Lambda -> On-Prem

Lambda SSHs into the On-Prem server and runs the command: nohup python3 rules_wrapper.py --token TOKEN_XYZ... &. It disconnects immediately.

4

Execution

On-Premises

The legacy rules_loader.pl script runs locally, connecting to Oracle DB and uploading consolidated JSON to S3. This takes ~30 minutes.

5

The Callback

HTTPS API

Wrapper finishes. Calls AWS API: SendTaskSuccess(TOKEN_XYZ...). Step Functions resumes and triggers the Rules Processor.
