Strategic Pivot: Hybrid SSH Architecture

Policy Watch is transitioning to a Hybrid Control Plane architecture. Due to critical IP reputation risks with cloud-based scraping and complex legacy database dependencies, we are retaining core ingestion modules On-Premises. These will be invoked directly via Secure SSH Commands from AWS Step Functions, utilizing the "Callback Token Pattern" for robust status tracking without needing complex message queues.
