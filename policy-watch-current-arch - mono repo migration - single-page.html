Executive Summary

Transitioning Policy Watch from a hybrid On-Prem/ECS architecture to a fully Cloud-Native, Monorepo Architecture. The goal is to standardize tooling, eliminate legacy dependencies, and implement event-driven orchestration via AWS Step Functions.

  • 2 Automate via Step Functions
  • 3 Monorepo Consolidation
  • 4 Modernize Legacy Perl/Oracle

Moving scrapers to AWS Public IPs invites immediate blocking. Resolution: Implement a Commercial Proxy Rotation Service.

Rewriting Perl to Python carries high risk of logic bugs. Resolution: Implement "Shadow Mode" for 2 weeks before cutover.
