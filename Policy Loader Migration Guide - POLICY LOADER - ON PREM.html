Parse Token passed via SSH parser = argparse.ArgumentParser() parser.add_argument("--token", required=True) args = parser.parse_args() client = boto3.client('stepfunctions', region_name='us-east-1') try: # 2. Run Legacy Scraper print("Starting Policy Loader...") subprocess.run(["python3", "policy_loader_main.py"], check=True) # 3. Success Callback client.send_task_success( taskToken=args.token, output='{"status": "success"}' ) except Exception as e: # 4. Failure Callback client.send_task_failure( taskToken=args.token, error="WrapperError", cause=str(e) )

Connect to On-Prem Server ssh = paramiko.SSHClient() ssh.connect(os.environ['ON_PREM_IP'], pkey=get_key()) # 3. Fire and Forget (nohup) cmd = f"nohup python3 wrapper.py --token '{token}' &" ssh.exec_command(cmd) return {"status": "Command Sent"}

Operational Flow: "Life of a Job"

1

08:00 AM Trigger

EventBridge fires "Start Execution". Step Functions wakes up.

2

Token Generation

Step Functions creates Token AAAA-1234... and pauses execution (Wait State).

3

The Handoff

Lambda SSHs into On-Prem server and runs wrapper.py --token AAAA... in background.

4

Execution (On-Prem)

Legacy policy_loader_main.py runs for ~45 minutes scraping PDFs.

5

The Callback

Wrapper finishes. Calls AWS API: SendTaskSuccess(AAAA...). Step Functions resumes.
